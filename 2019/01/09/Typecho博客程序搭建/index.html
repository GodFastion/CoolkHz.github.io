Typecho博客程序搭建

发表于 2019-01-09 | 分类于 教程 | | 阅读次数:

适合新手小白

简单几步，搭建属于你自己的blog

准备工作：

  • 虚拟主机或者服务器

  • 域名

一、虚拟主机

搭建博客的话

还是虚拟主机经济实惠些

所以这里我用的是虚拟主机

和上次一样

我这次演示的还是这家的主机

https://idc.xsazz.com

购买主机

二、下载博客程序

先发一个轻松搭建的博客程序

点击下载

Typecho官网

官网下载的压缩包需要改成.zip

三、域名解析

推荐一个二级域名解析网

http://dns.nyyd.net

如果可以的话推荐自己购买一个域名

我这里就用二级做演示了

具体操作都差不多

打开你主机的控制面板

点击域名绑定

不同的主机控制面板不同

换汤不换药

按照提示绑定域名

推荐使用A记录解析

有的免费主机

可能不支持A记录解析

CNAME解析方法和A相同

按主机的域名绑定说明就可以

域名解析完了

接下来主机绑定域名

域名绑定完毕

四、安装博客程序

找到刚才下载的压缩包

上传到根目录

并解压

解压密码为面板登陆密码

打开控制面板首页

将php版本更改为7.2.1

点击确定

访问刚才解析的域名

会自动跳转到安装页面

点击下一步

按照主机控制面板右侧

填写数据库信息

注意区分”数据库名“和”数据库用户名“

填写完毕点击 确认，开始安装》

等待安装成功

安装完成

接下来就可以正常访问你的博客了

你的域名后面加 /admin

则进入博客后台控制面板

恭喜你 已经有一个属于自己的博客了

快去写下第一篇文章吧！
